About NASA Space Explorer

A modern, responsive web application built to make space exploration data accessible to everyone using NASA's Open APIs.

APIs Used

  • ✓ APOD (Astronomy Picture of the Day)
  • ✓ Mars Rover Photos API
  • ✓ NeoWs (Near Earth Object Web Service)
  • ✓ DONKI (Space Weather Database)

Tech Stack

  • Backend: Python, Flask, SQLAlchemy
  • Frontend: HTML5, CSS3, JS, Bootstrap 5
  • Design: Glassmorphism, Space Theme
